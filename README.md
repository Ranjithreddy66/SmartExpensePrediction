# Smart Expense Prediction Using Machine Learning

## Overview

A Machine Learning project that predicts the next month's expenses based on historical spending patterns.

## Features

- Dataset Generation
- Exploratory Data Analysis
- Data Visualization
- Data Preprocessing
- Multiple ML Algorithms
- Model Comparison
- Hyperparameter Tuning
- Feature Importance
- Expense Prediction

- Pandas
- NumPy
- Matplotlib
- Scikit-learn

## Algorithms Used

- Decision Tree Regressor
- Random Forest Regressor
- Gradient Boosting Regressor
